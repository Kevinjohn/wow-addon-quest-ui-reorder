# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its code of conduct. The full text is available at that link.

In short: everyone interacting in this project's issues, pull requests, and
other spaces is expected to be respectful, welcoming, and constructive.
Treat others as you'd like to be treated.

## Enforcement

If you experience or witness behaviour that falls short of that standard,
please report it privately to **email@kevinjohngallagher.com**. Reports will be
reviewed and handled discreetly and fairly.
